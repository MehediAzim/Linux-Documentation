## 1. How to reduce terminal text / Customizing BASH Prompt

#### Step 1: Small
```console
rafsanjani@mrz:~$ PS1='[\u@\h \W]\$ '
```
#### Step 2: Extra Small
```console
rafsanjani@mrz:~$ PS1='[@ \W]\$ '
```

## X. Regular Expression

#### Step 1: Search a keyword from a file
```console

```
